# openmm8.4-AMOEBA
 
OpenMM 8.4 built from source for CUDA 12.2.
 
## Purpose
 
The `conda-forge` build of OpenMM does not run on older CUDA drivers. This image compiles OpenMM from source against CUDA 12.2. It is intended for polarizable molecular dynamics simulations using the AMOEBA force field via OpenMM.
